# Redmi



## Deskripsi Proyek

Proyek ini adalah portofolio sederhana untuk belajar semantic HTML dan CSS.

Tujuannya adalah membuat halaman web yang rapi, mudah dibaca, dan sesuai praktik terbaik HTML5.



## Struktur Proyek

- `index.html` - halaman utama dengan konten profil dan informasi diri.

- `style.css` - gaya tampilan untuk layout, warna, dan responsivitas.

- `script.js` - logika JavaScript sederhana untuk interaksi.

- `image/` - folder untuk aset gambar yang digunakan di halaman.



## Fokus Pembelajaran

- Menggunakan elemen semantic seperti `header`, `main`, `section`, `article`, `footer`.

- Membuat navigasi yang jelas dengan `nav` dan `ul`.

- Memperhatikan aksesibilitas dengan `aria-label` dan `alt` pada gambar.

- Menyusun konten agar lebih mudah dipahami oleh mesin pencari dan pengguna.



## Fitur Utama

- Header dengan judul dan menu navigasi.

- Bagian profil dan info diri.

- Daftar project (dapat dikembangkan lebih lanjut).

- Footer sederhana.



## Catatan Belajar

Proyek ini cocok untuk pemula yang ingin memahami bagaimana membuat struktur HTML yang baik dan menata tampilan dengan CSS.



## Note yang Harus Di-Fix

- Perbaiki tampilan menu agar tetap terlihat ketika halaman dibuka di perangkat mobile.

- Gunakan semantic tag lebih konsisten, seperti `section` untuk area konten dan `article` untuk blok informasi.

- Tambahkan teks alternatif (`alt`) yang lebih deskriptif pada semua gambar.

- Periksa penggunaan atribut `aria-label` agar jelas dan membantu screen reader.

- Susun ulang konten project agar lebih mudah diakses dan dinavigasi.

- Tambahkan responsivitas untuk tampilan desktop dan mobile.



## Ide Pengembangan Selanjutnya

- Tambahkan section `projects` dengan kartu-kartu project lengkap.

- Gunakan CSS Grid atau Flexbox untuk layout responsif.

- Perbaiki tampilan menu agar terlihat di semua perangkat.

- Tambahkan animasi halus untuk interaksi pengguna.
